# Security Policy

## Supported Versions

| Version   | Supported          |
| -------   | ------------------ |
| >= 0.0.49 | :white_check_mark: |
|  < 0.0.49 | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities directly to
[Broad Institute Wintergreen Team](mailto:wintergreen@broadinstitute.org).
